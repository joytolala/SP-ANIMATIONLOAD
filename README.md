## PAGE LOAD ANIMATION for SmartPhone
スマートフォン用のページロードアニメーション

## Discription
- JSはHTML内の最後に記述するだけ 
- body,a,submit,formにクラスを追加でカンタンに 
- CSSに記述するのは.transitionだけ 

## Effect
### In Effect
-  fadeIn
-  scaleFromOver
-  scaleFromSmall
-  slideToLeft
-  slideToRight
### Out Effect
-  fadeOut
-  scaleToOver
-  scaleToSmall
-  slideToLeft
-  slideToRight
